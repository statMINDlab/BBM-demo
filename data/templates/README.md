# Templates
All templates used in the companion paper are available either directly in this repository or after signing the relveant Data User Agreement (DUA). A detailed list including sources and adaptations is presented here:

## PROFUMO
Presented in _Modelling subject variability in the spatial and temporal characteristics of functional modes_ (Harrison et al. 2020), [doi:10.1016/j.neuroimage.2020.117226](https://doi.org/10.1016/j.neuroimage.2020.117226) can be downloaded from the study [repository](https://balsa.wustl.edu/study/gm40X), under the name `subject_maps_group12_run.dtseries.nii`.

After converting to dscalar, the subcortical structures where removed and the Human Connectome Project (HCP) medial wall mask was applied, the resulting template is saved as a RDS file and presented in this directory.

## MSC
An analogous process was performed for the MSC parcellation available for download from the companion [repository](https://github.com/MidnightScanClub/MSCcodebase/blob/master/Utilities/Conte69_atlas-v2.LR.32k_fs_LR.wb/Networks_template.dscalar.nii)

### HCP
HCP group ICA functional maps can be obtained from the HCP S1200 release, available within [ConnectomeDB](https://balsa.wustl.edu) tool after signing a Data User Agreement.
